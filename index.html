Шпаргалка до користуванням git

git config --global user.name blabla Особливі налаштування для встановленню зв'язку з віддаленим репозиторієм ввівши ім'я
git config --global user.email blabal@bla Особливі налаштування для встановленню зв'язку з віддаленим репозиторієм ввівши емейл
git init Команда ініціалізації git
git status Показує в якому статусі знаходяться файли в локальному репозиторії
git add Вказує гіту слідкувати за вказаними файлами
git commit Фіксує стан роботи в певній точці і добавляє комент щоб знати про що зміни
git log Виводить інформацію про попередні коміти
git checkout Дозволяє переключити на вибраний коміт або на гілку
git branch Дозволяє показати які зараз є гілки, а також створити нову гілку додавши ще один параметр
git push Закидує файли на віддалений репозиторій
git pull Дозволяє затягнути файл з віддаленого репозиторію до себе на локальний
git merge З'єднує дві гілки, залежно з якої гілки прописувати команду
git clone Копіюємо репозиторій на локальний сервер
.gitignore файл Це файл, в якому ми прописуємо назви файлів або папок, які не повинні бути запушені або закомічені
git reset [файл] Відміняє індексацію вказаного файлу, при цьому зберігає його вміст
git stash Тимчасово зберігає всі незафіксовані зміни відстежуваних файлів
